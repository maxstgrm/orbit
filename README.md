# 🌐 Orbit – Finde deine Leute

Eine mobile Web App zum Matchen mit potenziellen Freunden und Gruppen – kein Dating, echte Verbindungen.

## Features

- **Big Five Persönlichkeitstest** (OCEAN) – 10 Fragen, wissenschaftlich fundiert
- **Personen-Matching** – Ähnlichkeits-Score basierend auf OCEAN-Profil
- **Gruppen-Matching** – passende Gruppen nach Persönlichkeit und Interessen
- **Sternzeichen Modus** – optionale Elementkompatibilität (Feuer/Erde/Luft/Wasser) fließt in den Score ein
- **Chat** – Nachrichtenbereich mit Mock-Konversationen
- **Radar-Chart** – visuelle Darstellung des eigenen OCEAN-Profils

## Tech Stack

- Vanilla HTML / CSS / JavaScript (Single File)
- Kein Build-Step, keine Dependencies
- Mobile-first, PWA-ready

## Starten

Einfach `orbit-app.html` im Browser öffnen – fertig.
